Ultimate Sentiment Analyzer

Overview
Ultimate Sentiment Analyzer is a Python-based web application built using Streamlit that performs real-time sentiment analysis on user-provided text. The application classifies input text into Positive, Negative, or Neutral sentiment using Natural Language Processing (NLP) techniques.

This project demonstrates practical application of NLP concepts along with an interactive web interface, making it suitable for data analytics, machine learning, and software development roles.
Key Features
- Real-time sentiment analysis of user input text
- Interactive and user-friendly Streamlit interface
- Sentiment visualization using Plotly
- NLP processing using TextBlob and NLTK
- Lightweight and easy to deploy

Tech Stack
- Programming Language:** Python  
- Web Framework:** Streamlit  
- NLP Libraries:** TextBlob, NLTK  
- Data Handling:** Pandas  
- Visualization:** Plotly

Installation
Ensure Python 3.10+ is installed.
Install the required dependencies:
pip install -r requirements.txt

Download required NLTK/TextBlob data:
python -m textblob.download_corpora


How to Run the Application
Navigate to the project directory:
cd "C:\Users\”path of the file” "
Run the Streamlit application:
python -m streamlit run sentiment_app_cool.py
The application will automatically open in your default web browser.
